## <h2> 📝 Descrição do Projeto </h2>

O site de apresentação retrata o portfólio de uma desenvolvedora feito em **HTML** e **CSS** através de um molde no **Figma**. De novidades, o projeto tem mais duas novas abas também feitas em HTML usando as tags de navegação, semelhante aos menus das aplicações web do mercado. Além disso, o site conta com uma pasta só para assets e para estilos, seguindo as boas práticas de programação. <br>

Outro ponto de destaque dessa versão é o *hover* dos botões, onde ao usuário passar o cursor em cima do botão é alterada a cor dele. E, por fim, abaixo da aplicação há o rodapé do projeto contendo os créditos do autor, trabalhando as possíveis áreas onde um desenvovledor pode acessar, indo desde o *header* até o *footer*  <br>

Essa proposta foi realizada visando a aprovação e aprendizado prático do curso *HTML e CSS: Cabeçalho, footer e variáveis CSS* (link nos créditos no fim da documentação). <br>

## <h2> 🎯 Objetivos do Curso </h2>

- Entenda o dia a dia de um desenvolvedor Front-end de forma prática;
- Saiba como criar um código HTML e CSS seguindo boas práticas de programação;
- Descubra como utilizar variáveis CSS para manter seu código sustentável e limpo;
- Trabalhe com mais de uma página HTML no mesmo projeto;
- Conheça as pseudo classes com exemplos reais e práticos. <br>
